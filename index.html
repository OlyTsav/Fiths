Documentation

  1. Make any block element with the class of "container"
  2. Add as many child block elements with a class of "box"
  3. Fiths.js will automatically calculate box width relative to how many boxes are inside your container, and adjust to screen size accordingly.

<section class="container">
    <div class="box">Halves</div>
    <div class="box">Halves</div>
</section>

Some Notes

There is another div in my visual examples with a class of "boxcontent" for padding and additional syling (cursive font, padding, min-height, background-color, etc.) inside of each "box". This div is not shown in the code samples. Fiths provides the basic framwork and width sizing for responsive web design but leaves the rest of a sites styles up to the designer.
